📖 HK Quant Master 系統使用說明書

1. 系統大腦：狀態切換邏輯 (Regime Switching)

本系統的核心優勢在於「判斷大環境」。系統每天會掃描香港盈富基金 (2800.HK) 的價格與 200 日移動平均線 (200MA) 之間的關係：

  • 🟢 牛市狀態 (大盤 > 200MA)：系統啟動「海龜突破法」，專挑創下 20 日新高的強勢股，順勢追擊。
  • 🔴 熊市狀態 (大盤 < 200MA)：港股大部分時間處於此狀態。系統會關閉追高策略，啟動「RSI 超賣抄底法」，專門尋找被市場錯殺、跌穿布林通道下軌的股票進行低吸。

2. 面板資訊解讀 (如何看懂訊號)

在「實盤庫存」或「今日訊號」分頁中，您會看到強大的「基本面護城河」標籤：

  • 高股息防護 (綠色顯示)：當殖利率 > 6% 時會以綠色高亮。在熊市抄底時，高股息能為您提供強大的「下跌緩衝墊」，就算被套牢也能靠配息回本。
  • 業績動能確認：如果一檔股票暴跌，但業績顯示「強勁成長 (+)」，這就是黃金坑。若顯示「衰退中 (-)」，代表下跌是有合理原因的，強烈建議反彈即走，不可長抱。

3. 虛擬實盤基金 (Paper Trading) 運作機制

這是一個全自動運行的機器人公開基金：

  • 初始資金：預設為 $100,000 港幣。
  • 自動建倉：只要出現符合條件的新訊號，且帳戶可用現金大於 $10,000，系統就會自動動用約 $1 萬港幣買入該股票。
  • 無情平倉：系統每天會結算價格。如果股價跌穿「嚴格止損價(SL)」或突破「目標止盈價(TP)」，系統隔天就會自動賣出，記錄在歷史平倉區。

本網站託管於 GitHub Pages，透過 GitHub Actions 每天在 香港時間下午 4:30 (港股收盤後) 自動執行。您不需要做任何事，每天傍晚打開網頁，就能看到最新的實盤戰況與明日的交易計畫！
